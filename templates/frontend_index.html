Demo JS flows (from index.html)

The single-file demo includes tiny demo functions used from the browser console:

  • window.demoRegisterAndBook() — create a demo user and book a service via the API
  • window.demoCheckout() — create a PaymentIntent for your first order
  • window.demoCreateOrderAndCheckout() — create an order then open Stripe card modal

React app (development)

The React TypeScript homepage lives at templates/ubu-lite-homepage. To run it locally:

cd templates/ubu-lite-homepage; npm install; npm start

If the React dev server is not running, you can build the React app and serve it from Django at /react-app/

Build instructions (writes into templates/ubu-lite-homepage/build):

cd templates/ubu-lite-homepage
npm install
npm run build
# make sure the build/ directory exists in templates/ubu-lite-homepage

Notes

  • To view Stripe flows you must set STRIPE_PUBLISHABLE_KEY and STRIPE_SECRET_KEY in your environment.
  • The React app is a separate dev server; in production you can build it and serve files from Django static files.
